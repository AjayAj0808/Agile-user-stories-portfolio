Project: Fitness Tracker

User Story:
As a fitness enthusiast, I want to track my daily calorie intake so that I can achieve my fitness goals.

Acceptance Criteria:
User can input meals
System calculates calories
User can see progress

Project: Vehicle Maintenance

User Story:
As a vehicle owner, I want to receive maintenance reminders so that I can maintain my car properly.

Acceptance Criteria:
Reminders based on time or mileage
User can log service
System stores history

Project: Data Automation

User Story:
As a data analyst, I want to automate data tasks so that I can reduce manual work.

Acceptance Criteria:
Data processing is automated
Reports are generated automatically
Time is reduced
